| ------------------------ | ------------------------------ |
| IE 11                    | Not Support                    |

| Chrome 52+               | Chrome 63+                     |
| Opera 36+                | Opera 46+                      |
| Microsoft Edge 40.15063+ | All Support                    |
| Safari 11+               |                                |
| MacOS 10+                | _                              |
| Win 7+                   | _                              |
| Android 4.1+             | _                              |
| iOS 9.0+                 | _                              |


## 使用说明

- 修改 constraints 值进行 applyConstraints 限制，看浏览器是否支持applyConstraints 接口，支持的话是否applyConstraints 成功！
